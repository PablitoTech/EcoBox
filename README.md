# 🌱 EcoBox - MVP Open Source

EcoBox es una propuesta digital diseñada para facilitar el consumo responsable mediante un modelo de suscripción mensual. Este proyecto busca reducir la fricción en la adopción de hábitos sostenibles, permitiendo que hogares urbanos accedan a productos ecológicos de manera práctica y confiable.

## ⚠️ AVISO IMPORTANTE: Datos de Demostración

Este repositorio es un **Producto Mínimo Viable (MVP)** con fines de validación visual y de experiencia de usuario. Por tanto:

1.  **Datos Ficticios:** Todos los nombres de usuarios, testimonios, transacciones y métricas de impacto que aparecen en la aplicación son **puramente ficticios**. No representan clientes ni operaciones reales.
2.  **Imágenes Generadas por IA:** Todas las fotografías de productos, personas (equipo y usuarios) y entornos han sido **generadas mediante Inteligencia Artificial**. No se han utilizado imágenes de personas reales ni material protegido por derechos de autor de terceros.
3.  **Sin Transacciones Reales:** Las pantallas de pago y suscripción son simulaciones de flujo. No se procesan pagos reales ni se recolectan datos bancarios verdaderos.

## 🚀 Objetivo del Proyecto

EcoBox se concibe como un producto digital de estética moderna (tipo SaaS) que busca:
*   Generar conciencia sobre el impacto ambiental en el hogar.
*   Simplificar la transición hacia un estilo de vida más verde.
*   Validar la propuesta de valor de un modelo de "Caja de Suscripción" ecológica.

## 🛠️ Instalación y Uso

Sigue estos pasos para ejecutar el proyecto localmente:

### 1. Instalar dependencias
Desde la raíz del proyecto, ejecuta:
```bash
npm install
```

### 2. Ejecutar en modo desarrollo
Para iniciar la aplicación, ejecuta:
```bash
npm run dev
```
La aplicación estará disponible en `http://localhost:3000` (o el puerto que indique la consola).

### 3. Construcción para producción
```bash
npm run build
```
